Gastric Disease Diagnostic Expert System Using Naive Bayes Method and Certainty Factor Method

Description

This expert system for diagnosing gastric diseases was created to help speed up and make it easier for health center doctors to diagnose patients, and was also created for the wider community to carry out initial diagnoses of the gastric diseases they suffer from

Features

  • Can record patient data
  • Can record all diagnosis data
  • Can record the history of all patient diagnosis
  • Provide information for specific type of disease
  • Provide self diagnosis method
  • Provide directed diagnosis method
  • Provide indirect diagnosis method (with the help of proxy, e.g. worker beside doctor)
  • Provide direct diagnosis method (which involving patient and doctor directly)
  • Considering multiple variable to decide the result of system
  • Can dignose patient by considering existing data in database, patient symptoms,and certainty of each of that symptom and also by considering certainty of actual diagnosis that is done by the docters themselves
  • Printable/Exportable data for each type of data in the system
  • Printable diagnosis result
  • Easy to use user interface
